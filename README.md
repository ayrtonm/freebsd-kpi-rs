# FreeBSD KPI rust bindings

These are experimental bindings for writing FreeBSD kernel drivers in rust. They are intended for
use with the latest version of FreeBSD and rustc edition 2021. Note that since the FreeBSD KPI is a
moving target this repository may be broken on the FreeBSD-src tip of tree. For a known working
commit see [this branch](https://github.com/ayrtonm/freebsd-src/tree/m2).
